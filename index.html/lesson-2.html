WHAT IS VISUAL STUDIO CODE?

Visual Studio Code is a streamlined code editor with support for development operations like debugging, task running and versio control. It aims to provide just the tools a developer needs for a quick code-build-debug cycle and leaves more complex workflow to flutter featured IDEs (Integrated Development Environment)
