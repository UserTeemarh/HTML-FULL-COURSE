The <a> (anchor) Element

The <a> element with its href attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page, or anything else a URL can address, content within each <a> should indicate the link's destination.

The <a> (anchor) Reconigtion

  • An unvisited link is underlined and blue in color
  • A visited link is underlined and purple in color
  • An active link is underlined and red
ANCHOR <a> TAG ATTRIBUTES
ATTRIBUTES VALUE DESCRIPTION
href URL The href (hypertext reference) attribute specifies the URL of the page the link goes to. if the href attibute is not present, the <a> tag will not be a hyperlink.
target _blank
_parent
_self
_top 		The target attribute specifies where the linked document will open when the link is clicked. this will enable the linked document will open in a new tab or (older browser) a new window
title text The title attribute gives users an imporatant indication as to where they will be directed to when they click on a link. if the user hovers with mouser over a link reference, the title text, which was defined in the attribute, will be displayed.
download   The download attribute specifies that the target (the specified in the href attribute) will be downloaded when a user clicks on the hyperlink.
hreflang   The hreflang attibute specifies the language and geographical target of a webpage.
name   The name attribute specifies the name of for an <a> element.

Ways of implementing the Value of href attribute

Adding Phone Number Call link to a Website
tel: follow by the phone-number to be linked, this creates the call link. And this tells the browser how to use the number.
Adding Email link to a Website
  • Insert the mailto: into the href attribute follow by the email address.
  • To have the email sent to multiple email addresses, comma-seperate each email address
Adding SMS link to a Website
Insert the sms: into the href attribute follow by the phone-number of the recipient
Adding an Absolute URL link to a Website
Insert the correct URL of the page you want the content to be linked to into the href attribute
