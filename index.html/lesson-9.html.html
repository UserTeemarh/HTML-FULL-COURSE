THE FORM ELEMENT

<form> is an HTML element to collect input data containing interactive controls. It provide facilities to input text, number, email, password and other controls field. etc. Forms are generally used when you want to collect data from the user.

FORM ELEMENT AND DESCRIPTION
FORM ELEMENT DESCRIPTION
<label>..</label> It is used to associate a text label with a form <input> field. The label is used to tell users the value that should be entered in the associated input field
<input> It is used to specified an input field where the user can enter data. The <input> element is the most important form element. The <input> element can be displayed in several ways, depending on the type attribute
<select>..</select> The <select> element is used to create a drop-down list. It is used to collect user input.
<option>..</option> The <option> tag defines an option in a <select> tag. NOTE: The <option> tag can be used without any attributes, but you usually need the value attribute, which indicate what is sent to the server on form submission.
<optgroup></optgroup> It is used to group related options in a <select> element (drop-down list). It is used if there is a long list of options, groups of related options are easier to handle for a user
<textarea>..</textarea> It is used to collect unlimited multi-line text like comment or reviews. you can specify the size of a text area by the cols and rows attributes
<button>..</button> It defines a clickable button which submits all form values to a form-handler. The form-handler is typically a server page with a script for processing the input data. The form-handler is specified in the form's action attribute

FORM ATTRIBUTES

Form attribute specifies the form the element belongs to. The value of this attribute must be equal to the id attribute of a <form> element in the same document.
