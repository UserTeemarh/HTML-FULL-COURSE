THE TABLE ELEMENT

The <table> element represents tabular data __ that is, information presented in a two-dimensional table comprised of rows and columns of cells containing data.

Syntax:
The <table> tag is written as <table>...</table> with the various table element nested between the start and end tags.
TABLE ELEMENT AND DESCRIPTION
TABLE ELEMENT DESCRIPTION
<caption>..</caption> It identifies a table title. The <caption> tag must be inserted immediately after the <table> opening tag. NOTE: a table caption will be center-aligned above a table.
<thead>..</thead> The <thead> tag is used to group header content in an HTML table.
<th>..</th> The <th> Element defines a cell as the header of a group of table cells.
<tbody>..</tbody> The <tbody> tag is used to group the body content in an HTML table
<td>..</td> The <td> (table data) Element defines a cell of table that contains data. It participates in the table model
<tr>..</tr> The <tr> (table row) Element defines a row of cells in a table.
<tfoot>..</tfoot> The <tfoot> tag is used to group the footer content in an HTML table.
<colgroup>..</colgroup> It specifies a group of one or more columns in a table for formatting. It is useful for applying styles to entire columns, instead of repeating the styles for each cell, for each row
<col> It typically implemented as a child-element of a <colgroup> element. It can be used to target a column in an HTML table. This element must not contain any content, and does not need a closing tag.
