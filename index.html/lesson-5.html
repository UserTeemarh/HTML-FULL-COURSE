HTML ELEMENTs (2)

TAG DESCRIPTION
<p>...</p> It is used to define paragraphs. It is a block-level element, and it is commonly used to structure and format text on a webpage
<img> It is used to embed images in a webpage. It is an empty self-closing tag, and does not have a closing tag.
<video>...</video> It is used to embed videos in a webpage
<audio>...</audio> This tag is used to embed sound content in a document, such as music or other audio streams.
<a>...</a> An (anchor) element creates hyperlink to webpages or anything else a URL can address. content within each <a> should indicate the link's destination.
<br> It is used to insert a line break or line break-like spacing within text. It is an empty, self-closing tag and doesnot have a closing tag.
<hr> It is used to create a horizontal line between the two paragraphs. It is often used to visually seperate content or sections on a webpage.
<ul>...</ul> Represents an unordered list of items; usually a bulleted list by default, in which <li>(list item) must be contained inside of it.
<ol>...</ol> Represents an ordered list of items; usually a numbered list by default, in which <li>(list item) must be contained inside of it.
<style>...</style> Allows authors to put style sheet rules in the head of the document
<link>...</link> Signals that the HTML document uses another document, most commonly used to link to an external CSS file
<dt>...</dt> It is used to define a term in a description list<dl>. The term is typically followed by a description of that term, which is defined using <dt>(description definition) tag
<div>...</div> It defines a division or a section in an HTML document. div element is used to group other element of the page together.
<span>...</span> It is an inline element that is typically used to apply styles to a specific section of text within a larger text block.
<figure>...</figure> It specifies self-contained content. the content of the <figure> element is related to the main flow. its posiition is independent of the main flow.
<figcaption>...</figcaption> It defines a caption for a <figure> element. the <figcaption> can be placed as the first or last child or the <figure> element
<fieldset>...</fieldset> The Fieldset tag is used to group element as a form. It creats a border around its element
<legend>...</legend> A legend tag defines the caption or title for the <fieldset> tag, and it is the element's first child. Because it is a caption, it normally sits on top of the frame
