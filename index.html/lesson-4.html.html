HTML Element

!DOCTYPE
The HTML document type declaration, also known as DOCTYPE. is the first-line of code required in every HTML document. The DOCTYPE declaration is an instructions to the web browser about what version of HTML the page is written in this ensures that the web page is parsed the same way by different web browsers
Html
the html element is the second-line of code required in every HTML document in which all html element is going to be wrapped in it. HTML has an attribute of lang: the lang attribute specifies the language of the element's content
Head
The <head> element is used to define the head portion of the document. This section typically contains meta-information about the HTML document, such as the title, character set, styles, script, and other metadata.
metadata
The metadata tags are used to provide information about the document itself. These tags are typically placed within the <head> section of an HTML document.

These metadata tags help browsers, search engines, and other tools interpret and display the HTMl document correctly and efficiently. They play a crucial role in aspects like character encoding, viewport settings, styling and external resource linking.

Metadata types and their attribute

<meta charset="UTF-8">
setting up the character set
Attribute description
Charset Charset stands for character set. It refers to a specific collection of characters that are associated with a numerical code, allowing computers to represent and interpret text.
The purpose is to specify the character encoding for the document. Character encoding defines the mapping between character codes and the actual characters they represent
Attribute value Description
UTF-8 UTF-8   8-bit unicode transformation format  is a character encoding that is capable of encoding all possible character, or code points, in Unicode.
Unicode is a standardized character encoding system that assigns a unique number(code point) to every character from every writing system in the world

specifying <meta charset="UTF-8"> ensure that browser interprets and displays the document using UTF-8 encoding, allowing for the correct representation of a wide range of characters.

< Meta http-equiv="X-UA-Compatible" content="IE-edge">
setting the http-equiv attribute
Attribute Description
http-equiv http-equiv stands for "HTTP(Hypertext Transfer Protocol)-Equivalent". It is used to define an HTTP header fo the information contained within the <meta> tag.The purpose of usings http-equiv is to provide metadata that emulates the behaviour of HTTP headers in an HTML document
Attribute value Description
X-UA-Compatible X-UA-Compatible is an HTTP header that is used to control the document mode and compatibility mode of Internet Explorer. It allows web developer to specify how Internet Explorer should render a webpage.
Attribute Description
Content The content attribute in the <meta> tag is used to provide information or values associated with a particular metadata directive.
IE-edge The IE-EDGE value is used to instruct Internet Explorer to use the highest available document mode and rendering engine. This setting is commonly used to ensure better compatibility with modern web standard
<meta name="viewport" content="width=device-width, initial-scale=1.0">
setting the viewport
Attribute value Description
Viewport The Viewport is used to control the layout and scaling of a web page on different devices.
It helps create a responsive design that adjusts to the screen size of the device providing a better user experience on both deskstop and mobile devices
attribute value description
  The width=device-width sets the width of the viewport to the width of the device screen. This ensures that the content will be displayed at the actual width of the device, helping in creating responsive designs.
The initial-scale=1.0 sets the initial zoom level when the page is first loaded. A value of 1.0 means that there is no initial zoom, and the page will be displayed at its normal scale. This crucial for ensuring that the layout and content are apporpriately sized on various devices, especially on mobile devices where users might zoom in by default.

Title
The Title tag is used to define the title of HTML document. it sets the title in the browser toolbar. it provides the title for the web page when it is added to favorites. it displays the title for the page in search engine results.
body
the Body tag defines the main content of an HTML document which displays on the browser. it contains all the content of an HTML document, such as headings, paragraphs, images, heading, tables, lists, link etc.
heading
The Heading tag is used to define the headings of a page. they are six levels of headings defined by HTML. these 6 heading element are from h1-h6. h1 defines the most important heading, while h6 defines the least important heading. NOTE: only use one h1 per page- this should represent the main heading/subject for the whole page
