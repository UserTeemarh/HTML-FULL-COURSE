HTML FORMATTING ELEMENTS

HTML Formatting is a process of formatting text for better visualization to text content in HTML document. HTML provides us the ability to format text without using CSS.

HTML FORMATTING ELEMENTS
FORMATTING ELEMENT DESCRIPTION
<strong>...</strong> It is used for indicating an important or urgent text.
<mark>...</mark> It is used to highlight text that is of special interest or relevance in an HTML document. browsers render text within <mark> tag as text with a yellow background-color.
<u>...</u> <u>: The Unarticulated Annotation (underline) element. It is a length of inline text that stylistically looks different from its surronding text but has non-textual annotation. the default style for this element is a single underline
<del>...</del> <del>: The Deleted Text Element. It represents a range of text that has been deleted from a document but retained to show the history of modification made to the document
<ins>...</ins> <ins>: The Inserted Element. It is used to specify a block of inserted text. It is typically used to mark a range of text that has been added to the document. The inserted text is rendered as underlined text by the web browsers.
<sup>...</sup> <sup>: The Superscript Element. It is used to add a superscript text to the document. Superscript text appears half a character above the normal line and is rendered in a smaller font.
<sub>...</sub> <sub>: The Subscript Element. It is used to add a subscript text to the document. Subscript text appears half a character below the normal line and is rendered in a smaller font.
<strike>...</strike> <strike>: The Strike Element. It gives text the appearance of a strikethrough which draws a horizontal line over the text.
<small>...</small> <small>: The Small Element. It is used to set small font size, it decreases the font size by one size. i.e (from medium to small, from x-large to large)
<abbr>...</abbr> <abbr>: The Abbreviation ELement. It defines an abbreviation or an acronymn. The title attribute needs to be included in abbr opening tag to show the description for the abbreviation/acronym when you mouse over the text
<address>.</address> <address>: The Address Element defines the contact information for the author/owner of a document.
<em>...</em> It is used to define emphasized text. the content inside is typically displayed in italic
