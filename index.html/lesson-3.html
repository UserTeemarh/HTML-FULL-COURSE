INTRODUCTION TO HTML

HTML stands for "hypertext Markup Language". HTML is the standard markup language for document designed to be displayed in a web browser. It defines the meaning and structure of web content. It is often assisted by technologies such cascading style sheet(CSS) and scripting language such as JAVASCRIPT

components of HTML

HTML Element.
An HTML Element is a component of an HTML document that tells a web browser how to structure and interpret a part of the HTML document. HTML elements can contain formatting and instructions, semantic meaning, and content

There are two typical Types of HTML Element

Block-level Element
A Block-level Element creates a new section on the page. by default, a block-level element stretches to fill 100% of the viewport (i.e, the visible area of the webpage) and always starts a new line
Inline-level Element
An Inline-level Element exists in line with their parent element. both width and height of an Inline-level element are set by its content, also, Inline-level elements do not start on a new line. meaning: you can put multiple of them on the same line
HTML Tags
An HTML tag is a piece of markup language used to indicate the beginning and end of an HTML element in an HTML document. As part of an HTML element, HTML tags help web browsers convert HTML documents into web pages.
Many tags, though not all, use an opening tag and closing tag to wrap around the content that they are used to modify. Closing tags are denoted with a backslash like this: </tag_name>. HTML tags are not visible in the browser.
HTML attribute
An HTML attribute is a piece of markup language used to adjust the behavior or display of an HTML element. Attributes are used by including them in an opening HTML tag:
HTML Value
Value specifies the name of an attribute

Features of HTML

  • HTML helps to build structure of a website and is a widely used to Markup Language
  • Every Browser supports HTML
  • It integrates easily with other languages such as JAVASCRIPT and CSS
  • HTML has many tags and attributes which can shorten your line of code
  • HTML also allows utilization of templates, which makes designing a webpage easy
