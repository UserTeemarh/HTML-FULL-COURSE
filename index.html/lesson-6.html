“IMAGE,VIDEO AND AUDIO ELEMENT”

IMAGE <img> TAG
The <img> is used to embed an image in an HTML page. images are not technically inserted into a web page; images are linked to webpages. the <img> tag creates a holding space for the referenced image.

Image <img> tag syntax

  • create a new folder (inside the same folder that your HTML is opened with) and save it as img
  • copy and paste your image into the img folder you created.
  • then: declare the img tag and include the src attribute.
  • copy the image URL into the src attribute
IMAGE <img> TAG ATTRIBUTES
ATTRIBUTES VALUE DESCRIPTION
src URL the src attribute identifies an image by a URL. the image defined by the URL is retrieved by the browser and inserted into the document when the page loads.
alt text It specifies an alternate text for an area, if the image cannot be displayed. the alt attributes provides alternative information for an image if a user for some reason cannnot view it (because of slow connection, an error from the src attibute, or if the user uses a screen reader)
height pixels the height attibute specifies the height of an image, in PIXELS.
width pixels the width attibute specifies the width of an image, in PIXELS.
loading eager
lazy		 the loading attribute can be used to instruct the browser to defer loading of images that are off-screen until the user scrolls near them
title text the title attribute adds a tooltip with title text to the image
longdesc text the longdesc attribute in html is a URL, the target of which contains a long description of the non-text content. Authors can provide a description for an image by including text in a separate resource or within the text of the page containing the image.
THE VIDEO <video>....</video> ELEMENT
The <video> element embeds a media player which supports video playback into the document. It contains one or more <source> tags with different video sources. the browser will choose the first source it supports. The text between <video> and </video> tags will only be displayed in browsers that do not support the <video> element.

VIDEO <video>....</video> TAG SYNTAX

  • Create a new folder (inside the same folder that your HTML is opened with) and save it as video
  • The video to be used must be mp4
  • copy and paste the video into the video file you created.
  • Then: declare the video tag and add other necessary attribute to be able play the video on the browser
+
VIDEO <video>....</video> TAG ATTRIBUTES
ATTRIBUTES VALUE DESCRIPTION
src URL The src attribute specifies the location (URL) of the video file. The attribute uses ogg file on firefox, opera and chrome browsers. to play the video in old Internet Explorer and Safari, we must use an MPEG4 file.
controls controls Controls attribute specifies that the video player will display its media controls. These controls allow users to interact with features like play, pause, track, volume
autoplay autoplay Autoplay attribute is a boolean attibute that can be added to compel the browser to play the video automatically as soon as possible without waiting for the page to finish loading the data
muted muted Muted attribute is used in video player to tell the browser that playback should begin without audio. The muted attribute is most often paired with autoplay.
height pixels The height attibute specifies the height of an video player, in PIXELS.
width pixels The width attibute specifies the width of a video player, in PIXELS.
loop loop Loop attribute is a boolean attribute. when present, it specifies that the video will start over again, every time it is finished.
poster URL Poster attribute specifies an image to be shown while the video is downloading, or until the user hits the play button. if this is not included, the first frame of the video will be used instead.
preload auto
metadata
none		 Preload attribute lets the developer advice the browser how much of the video should be uploaded prior to viewing.
THE AUDIO <audio>....</audio> ELEMENT
The <audio> element is used to embed sound content in a document, such as music or other audio streams. the <audio> tag contains one or more <source> tags with different audio sources. the browser will choose the first source it supports.

steps for adding <audio> to HTML

  • choose an audio file in formats like mp3, WAV, or OGG
  • insert the <audio> element in your HTML, specifying the audio file with the src attribute
  • use the controls attribute to add player controls
AUDIO <audio>....</audio> TAG ATTRIBUTES
ATTRIBUTES VALUE DESCRIPTION
src URL specifies the URL of the audio file
autoplay autoplay specifies that the audio will start playing as soon as the page loads.
controls controls specifies that audio controls should be displayed (such as play/pause, button etc.)
loop loop specifies that the audio will start over again, every time it is finished
muted muted specifies that the audio output should be muted
preload auto
metadata
none		 specifies if and how the author thinks the audio should be loaded when the page loads
